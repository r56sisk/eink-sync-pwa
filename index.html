E-Ink Display Sync

First-time setup
  1. In Google Cloud Console: enable the Calendar API, create an API key + an OAuth Client ID (type "Web application"), add this page's URL to the client's Authorized JavaScript origins.
  2. Paste both values below. They persist in your browser — one-time setup.
  3. Reset the ESP32 so it's inside its 30s BLE window, then tap Sync Now.

If Google says "origin mismatch", the URL in the browser must exactly match an entry under Authorized JavaScript origins (no trailing slash, matching http/https).

Ready.
