First-time setup
  1. In Google Cloud Console: enable Calendar API, create an OAuth Client ID (type "Web application") with redirect URI https://developers.google.com/oauthplayground. Copy Client ID + Client Secret.
  2. Go to OAuth Playground → ⚙ gear → "Use your own OAuth credentials" → paste ID + Secret. Step 1: pick calendar.readonly → Authorize APIs → sign in → Allow. Step 2: Exchange authorization code for tokens. Copy the Refresh token.
  3. Paste Client ID, Client Secret, Refresh Token below. They persist in localStorage — one-time setup.
  4. Reset the ESP32 so it's inside its 30s BLE window, then tap Sync Now. No popups. Access tokens are auto-refreshed and cached for ~55 minutes.

See the comment at the top of this file's source for step-by-step details. The refresh token is long-lived — you enter it once. Revoke any time at myaccount.google.com/permissions.
