First-time setup
  1. In Google Cloud Console: enable the Calendar API, then create an API key and an OAuth Client ID of type "Web application".
  2. On that OAuth client, add:
    Authorized JavaScript origins: your hosting origin, e.g. https://you.github.io
    Authorized redirect URIs: the exact page URL, e.g. https://you.github.io/repo/pwa/ (trailing slash matters).
  3. Paste the API key + Client ID below. They persist in your browser — one-time setup.
  4. Reset the ESP32 so it's inside its 30s BLE window, then tap Sync Now. Google will redirect you for consent on first run; the token is cached for ~1 hour so subsequent syncs skip the redirect.

If Google says "redirect_uri_mismatch", the URL in your browser's address bar must match a value in Authorized redirect URIs exactly (protocol, host, path, and any trailing slash).
